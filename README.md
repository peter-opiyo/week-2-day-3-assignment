# Week 2 Day 3 Assignment: DOM Manipulation

This project contains three browser-based JavaScript apps built with HTML, CSS, and DOM manipulation.

## Apps Included

1. **Character Counter**
   - Real-time character count
   - Warning colour near the 280-character limit
   - Red error state after the limit
   - Submit button disabled when over the limit

2. **Dynamic Shopping List**
   - Add item name and quantity
   - Mark items as bought
   - Delete items
   - Remaining item counter
   - Items created with `document.createElement()`

3. **Simple Calculator**
   - Digits 0-9
   - Decimal support
   - Operators: +, -, ×, ÷
   - Clear and backspace buttons
   - Step-by-step chained calculations
   - Keyboard support bonus

## How to Run

Open any of these files with Live Server:

```text
counter.html
shopping-list.html
calculator.html
```
